Tactical Packet Analysis Helper

About

This app was created for the presentation "Tactical Traffic Analysis" at SANS Europe ICS Summit, 2024.

Its purpose is to demonstrate the possibility to connect multiple freely available tools into a powerful automation framework using data pipelines.

The author provides no warranties. The app has bugs, has not been tested at scale, and has obvious security issues which mean it should not be deployed in a production environment.

With that out the way, have fun and enjoy.
